public Int32 Read(Byte[] buffer, Int32 offset, Int32 count)

Parameters

buffer

Byte[]

An array of bytes. When this method returns, the buffer contains the specified byte array with the values between offset and (offset + count - 1) replaced by the bytes read from the cache.

offset

Int32

The zero-based byte offset in the buffer at which to begin storing the data read from the cache.

count

Int32

The maximum number of bytes to read from the cache.

Returns

The total number of bytes read into the buffer.
