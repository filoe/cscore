Properties

DelayTimes

Gets the caller provided delay time array, which receives delays for each destination channel in milliseconds. This array must have at least DstChannelCount elements. X3DAudio doesn't actually perform the delay. It simply returns the coefficients that may be used to adjust a delay DSP effect placed in the effect chain. This won't be calculated if the Delay flag is not specified when calling X3DAudioCalculate.

public Single[] DelayTimes { get; set; }

DopplerFactor

Gets the doppler shift factor. Scales the resampler ratio for Doppler shift effect, where: effectivefrequency = DopplerFactor × originalfrequency. Only calculated if the Doppler flag is specified when calling X3DAudioCalculate. When using X3DAudio with XAudio2 the value returned in the DopplerFactor would be applied to a source voice with SetFrequencyRatio.

public Single DopplerFactor { get; }

DstChannelCount

Gets the number of source channels.

public Int32 DstChannelCount { get; set; }

EmitterToListenerAngle

Gets the emitter-to-listener interior angle, expressed in radians with respect to the emitter's front orientation. Only calculated if the EmitterAngle flag is specified when calling X3DAudioCalculate.

public Single EmitterToListenerAngle { get; }

EmitterToListenerDistance

Gets the distance in user-defined world units from the listener to the emitter base position.

public Single EmitterToListenerDistance { get; }

EmitterVelocityComponent

Gets the component of emitter velocity vector projected onto emitter-to-listener vector in user-defined world units per second. Only calculated if the Doppler flag is specified when calling X3DAudioCalculate.

public Single EmitterVelocityComponent { get; }

ListenerVelocityComponent

Gets the component of listener velocity vector projected onto the emitter->listener vector in user-defined world units per second. Only calculated if the Doppler flag is specified when calling X3DAudioCalculate.

public Single ListenerVelocityComponent { get; }

LPFDirectCoefficient

Gets the LPF direct-path coefficient. Only calculated if the LpfDirect flag is specified when calling X3DAudioCalculate. When using X3DAudio with XAudio2 the value returned in the LPFDirectCoefficient member would be applied to a low pass filter on a source voice with SetFilterParameters.

public Single LPFDirectCoefficient { get; }

LPFReverbCoefficient

Gets the LPF reverb-path coefficient. Only calculated if the LpfReverb flag is specified when calling X3DAudioCalculate.

public Single LPFReverbCoefficient { get; }

MatrixCoefficients

Gets the caller provided array that will be initialized with the volume level of each source channel present in each destination channel. The array must have at least (SrcChannelCount × DstChannelCount) elements. The array is arranged with the source channels as the column index of the array and the destination channels as the row index of the array.

public Single[] MatrixCoefficients { get; set; }

ReverbLevel

Gets the reverb send level. Only calculated if the Reverb flag is specified when calling X3DAudioCalculate.

public Single ReverbLevel { get; }

SrcChannelCount

Gets the number of source channels.

public Int32 SrcChannelCount { get; set; }
